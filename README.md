PowerShell Scripts

A collection of beginner PowerShell scripts built while learning the fundamentals.

1. Device Name Validator:
This script asks the user to enter a comma-separated list of device names, then checks which names are valid based on a naming convention rule.

2. Ticket Priority Analyser:
This script asks the user to enter a comma-separated list of ticket priorities, then counts how many of each priority appear and prints a summary.

3. Password Policy Checker:
This script asks the user to enter a password, checks it against a simple policy, and returns a rating (Strong / Okay / Weak). If the password fails any rules, it also prints clear reasons explaining what needs improving.
